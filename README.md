Gestore di progetti offline-first basato su Electron 38, React 19, Ant Design 5 e un dominio condiviso TypeScript/SQLite pensato per ambienti air-gapped o installazioni on-prem. Include API REST, CLI di seeding e pipeline di rilascio Windows portabile.
